Arranqué mi proyecto final del bootcamp: un acortador de URLs fullstack completo.

**Qué construí:**
- Sistema de acortamiento de URLs con enlaces personalizables
- Generación automática de códigos QR
- Sistema completo de autenticación y autorización
- Dashboard con analytics detallados: clicks, referrers, dispositivos, navegadores, OS
- Backend robusto con NestJS y TypeScript
- Base de datos relacional con MySQL y TypeORM

**El desafío técnico:**
Quería entender la complejidad de construir una herramienta así desde cero. No solo hacer que "funcione" - quería hacerlo bien.

Lo más interesante fue:
- Crear un **decorador personalizado en NestJS** para manejar redirecciones
- Implementar **interceptors** para capturar y procesar requests
- Diseñar un sistema de **analytics** que registre todo sin impactar performance
- Arquitectura de **middlewares** para validaciones y seguridad

Este proyecto me enseñó más que cualquier tutorial. Enfrenté problemas reales, tomé decisiones de arquitectura, y lo llevé a producción.

**Duración:** Junio - Diciembre 2024  
**Resultado:** Un proyecto del que estoy genuinamente orgulloso.
