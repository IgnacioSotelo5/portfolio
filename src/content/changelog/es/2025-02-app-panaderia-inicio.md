Empecé a desarrollar una aplicación móvil para digitalizar las operaciones de la panadería familiar.

Lo que comenzó como un proyecto para aprender React Native se convirtió en algo más grande: una herramienta real que resuelve problemas reales.

**Qué estoy aprendiendo:**
- Mobile development con React Native y Expo
- El routing en mobile es completamente distinto a web (stacks, screens, tabs)
- Comunicación eficiente con backend en contextos móviles
- Optimización de listas y performance en dispositivos
- MongoDB con Prisma ORM
- Autenticación segura con tokens almacenados en `expo-secure-store`

La arquitectura está diseñada pensando en escalabilidad futura hacia web. Modular, separada por concerns, lista para crecer.

**Por qué importa:**
Este proyecto me obligó a pensar en UX para usuarios no técnicos (mi familia), optimizar para conexiones inestables, y diseñar pensando en el mundo real, no en tutoriales.

Es trabajo en progreso, pero cada feature que implemento me enseña algo nuevo.
