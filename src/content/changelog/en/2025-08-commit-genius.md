It's a tool that analyzes code changes (`git diff`) and automatically generates commit messages following Conventional Commits.

**What I learned:**
- Integrating AI APIs (Groq)
- Crafting effective prompts
- Modern backend with ElysiaJS
- How AI can automate repetitive tasks
